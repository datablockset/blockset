# Data Block Set

Data Block Set

## For Developers

Installation:

```
cargo install --git https://github.com/datablockset/blockset
```

Uninstall the `blockset`:

```
cargo uninstall blockset
```

### Commands

- address validation:
  ```
  blockset validate 3v1d4j94scaseqgcyzr0ha5dxa9rx6ppnfbndck971ack
  ```
- calculate address:
  ```
  blockset address ./README.md
  ```
